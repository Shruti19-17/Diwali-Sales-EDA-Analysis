# 🎇 Diwali Sales EDA Analysis

## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on Diwali sales data to uncover customer demographics, purchasing behavior, and product preferences.  
The goal is to generate actionable insights that can help businesses improve **targeted marketing strategies** and **boost sales performance** during festive seasons.

---

## 🛠️ Tech Stack
- **Python** 🐍  
- **Pandas** for data manipulation  
- **NumPy** for numerical operations  
- **Matplotlib & Seaborn** for data visualization  
- **Jupyter Notebook** for analysis workflow  

---

## 📊 Key Insights
- Identified **top-selling product categories** during Diwali.  
- Analyzed **regional sales trends** across different states.  
- Explored **customer demographics** (age, gender, occupation).  
- Highlighted **high-value customer segments** driving revenue.
